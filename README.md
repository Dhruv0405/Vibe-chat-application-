### Setup .env
```js

MONGODB_URI=...
PORT=5001
JWT_SECRET=...

CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...

NODE_ENV=development
```
#Build The App
```shell
npm run build
```


###Start the app
```shell
npm start
```
